Verifying a MyRecoveryPal Attendance Report

A one-page guide for probation officers, drug court coordinators, and public defenders. Verification takes under a minute and requires no account.

What you receive

A defendant using MyRecoveryPal Court Compliance can email you a PDF meeting-attendance report directly from the platform. Each report contains:

  • Defendant name, case number, court, and supervising officer
  • Every logged meeting: program (AA, NA, SMART Recovery, etc.), date, time, and location
  • A summary against the required attendance rate (e.g. 3 meetings/week)
  • A signed attestation block and a SHA-256 document fingerprint

Entries that are self-reported (no chair attestation) are explicitly flagged in the report — the document never claims more certainty than it has.

How to verify a report in 3 steps

1
Find the document fingerprint printed on the report (a 64-character code on the final page).
2
Go to the public verification page:
myrecoverypal.com/verify/court/<fingerprint>/
3
The page confirms whether a report with that exact fingerprint was generated by MyRecoveryPal, and when. If even one character of the PDF was altered after generation, the fingerprint will not match.
Why this is stronger than a paper card: a signature card can be forged or pre-signed; there is no way to check it later. The fingerprint is computed from the exact bytes of the PDF, so any edit — a date, a meeting count, a name — produces a mismatch you can detect independently of the defendant.

Privacy by design

The public verification page confirms only that a document with that fingerprint exists and when it was generated. It never displays the defendant's name, case number, or any personal information — so the verification link can be shared or filed without creating a privacy exposure.

Program-neutral by design

The platform tracks AA, NA, CA, MA, GA, SMART Recovery, Refuge Recovery, LifeRing, and other secular programs equally, consistent with appellate rulings that attendance mandates cannot be limited to 12-step programs.
