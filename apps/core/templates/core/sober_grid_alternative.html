What Happened to Sober Grid? (2024 Update)

Sober Grid is no longer available. The popular recovery social network shut down in 2023–2024 and was removed from both the App Store and Google Play. Here's the full timeline:

  • October 2021: Sober Grid founder Beau Mann tragically passed away at age 38.
  • 2022–2023: Without its founder, the platform struggled to maintain operations and funding.
  • 2023–2024: Sober Grid was unpublished from app stores and all services were discontinued.
  • 2024–Present: The Sober Grid app and website are no longer accessible. Previous user data is no longer available.

If you're searching for "is Sober Grid still available" or "Sober Grid app discontinued" — the answer is no, Sober Grid is permanently shut down. But the need for a recovery social network hasn't gone away.
