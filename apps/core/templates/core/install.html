Installing on Android

1

Visit MyRecoveryPal.com

Open Chrome or your preferred browser and navigate to www.myrecoverypal.com

2

Look for the Install Banner

After visiting the site, you'll see a banner at the bottom of the screen saying "Install MyRecoveryPal". Tap the Install button.

Screenshot: Install banner appearing at bottom of screen
3

Alternative: Use Browser Menu

If you don't see the banner, tap the menu button (three dots) in the top-right corner, then select "Add to Home Screen" or "Install App".

4

Confirm Installation

A popup will appear asking you to confirm. Tap Install or Add.

5

Find the App Icon

MyRecoveryPal will now appear on your home screen just like any other app. Tap it to open!

Installing on Firefox

Note: Firefox has different PWA support depending on your device.

Firefox for Android

1

Visit MyRecoveryPal.com

Open Firefox for Android and navigate to www.myrecoverypal.com

2

Open the Menu

Tap the three-dot menu button (⋮) in the top-right corner of Firefox.

3

Select "Install"

Look for the "Install" option in the menu and tap it.

Screenshot: Install option in Firefox menu
4

Confirm Installation

A popup will appear asking you to confirm. Tap Add or Install.

5

Find the App Icon

MyRecoveryPal will now appear on your home screen. Tap it to open!

Firefox for Desktop

Limited Desktop Support

Firefox desktop currently has limited PWA support. For the best experience on desktop, we recommend using:

  • Google Chrome - Full PWA support
  • Microsoft Edge - Full PWA support
  • Safari (on macOS) - Good PWA support

Alternative: Create a Bookmark

While Firefox desktop doesn't support full PWA installation, you can:

  • Bookmark MyRecoveryPal (Ctrl+D / Cmd+D)
  • Pin the tab for quick access
  • Use the site normally in your browser

Consider switching to Chrome or Edge for the full app experience with offline support and push notifications.

Installing on iPhone/iPad

1

Open Safari

Important: You must use Safari browser on iOS. Chrome and other browsers don't support app installation on iOS.

Navigate to www.myrecoverypal.com

2

Tap the Share Button

Look for the Share button at the bottom of Safari (square with an arrow pointing up) and tap it.

Screenshot: Safari share button location
3

Select "Add to Home Screen"

In the share menu, scroll down and tap "Add to Home Screen". You may need to scroll down to find this option.

Screenshot: Add to Home Screen option in share menu
4

Name the App

You'll see a preview of the app icon and name. You can edit the name if you'd like, or keep it as "MyRecoveryPal".

5

Tap "Add"

Tap the Add button in the top-right corner. The app will now appear on your home screen!

2

Click the Install Icon

Screenshot: Install icon in Chrome address bar
3

Or Use the Install Banner

Alternatively, click the Install button on the banner that appears at the bottom of the page.

4

Confirm Installation

MyRecoveryPal will open in its own window and appear in your applications menu. You can pin it to your taskbar or dock for quick access!

Troubleshooting

I don't see the install button

Try visiting the site a few times. Most browsers require 2-3 visits before showing the install prompt. Also make sure you're using a supported browser (Chrome, Edge, Safari on iOS).

How do I uninstall the app?

Android: Long-press the app icon → Uninstall
iOS: Long-press the app icon → Remove App
Desktop: Right-click the app icon → Uninstall

Will this use a lot of storage?

No! The app uses minimal storage (typically under 50MB) and automatically manages cache size.
